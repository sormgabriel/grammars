Si Σ es un alfabeto y n ∈ Z+, definimos las potencias de Σ recursivamente de la siguiente manera:

  1. Σ1 = Σ
  2. Σn+1={xy | x ∈ Σ, y ∈ Σn}
    3. Entonces, dado Σ = {0,1},
    para Σ1={x | x ∈ Σ} = {0,1},
    para Σ2={xy | x ,y ∈ Σ} = {00, 01, 10,11},
    para Σ3={xyz | x,y,z ∈ Σ} = {000,001,010,011,100,101,110,111},
    dado Σ = {a,b,c},
    para Σ1={x | x ∈ Σ} = {a,b,c},31
    para Σ2={xy | x ,y ∈ Σ} = {aa, ab, ac, ba, bb, bc, ca, cb, cc}, 32
    para Σ3={xyz | x,y,z ∈ Σ} = {aaa,aab,aac,aba,abb,abc, aca, acb, acc, baa, bab, bac, bba, bbb, bbc, bca, bcb, bcc,caa, cab, cac, cba, cbb, cbc, cca, ccb, ccc }, 33
